# homebrew-mae

Homebrew tap for [MAE — Modern AI Editor](https://github.com/cuttlefisch/mae).

## Install

```bash
brew tap cuttlefisch/mae
brew install mae
```

## GUI (optional)

```bash
brew install --cask mae
```

## Background daemon

```bash
brew services start mae
```

## Upgrade

```bash
brew upgrade mae
```

## Uninstall

```bash
brew uninstall mae
brew uninstall --cask mae   # if GUI installed
brew untap cuttlefisch/mae
```
